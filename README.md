"# fonoapi-ejs" 
Running sample of https://github.com/shakee93/fonoapi using ejs node js.

To run => 

run "nmp i" command

and change "fonoapi.token = 'your_token';" in app.js

then "node app.js"

Sample screenshot : https://ibb.co/jMbY6hz
